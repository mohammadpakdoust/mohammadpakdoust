## 🌟 Key Projects

### 🏥 [AI-Powered Hospital Route Manager](https://github.com/mohammadpakdoust/Datajam-2026) — *3rd Place (CGI DataJam 2026)*
*   Optimized triage datasets from Nova Scotia Health to validate data quality and identify routing inefficiencies.
*   Contributed to an AI-driven solution reducing patient wait times by an estimated **114 minutes**.
*   Developed rule-based guardrail testing frameworks and automated API test cases to secure safety of routing suggestions.

### 📄 [Document-Grounded Agentic RAG System](https://github.com/mohammadpakdoust/Document-Grounded-RAG-System)
*   Created a high-fidelity Retrieval-Augmented Generation (RAG) pipeline querying 10,000+ pages with **90%+ retrieval precision**.
*   Built with LangChain, ChromaDB, Gemini API, and Jina AI.
*   Integrated 3 key security & safety filters: PII detection (Microsoft Presidio), prompt injection classification, and LLM-as-a-judge faithfulness verification.

### 🔍 [Sparse Autoencoder (SAE) Mechanistic Interpretability](https://github.com/mohammadpakdoust/Qwen-Mechanistic-Analysis-Group2)
*   Trained a Sparse Autoencoder on the activations of the Qwen2.5-1.5B model to extract 6,144 interpretable features.
*   Conducted causal intervention experiments to prove the causal influence of specific numeric features on shifts in the model's outputs.

### 🤖 [GenAI Market Research Automation](https://github.com/mohammadpakdoust/GenerativeAI-Market-research)
*   Built a multi-agent automation pipeline (CrewAI, Gemini) that parallelizes web scraping, market analysis, and report compilation.
*   Reduced manual research time by **~3 hours per report**.

### 🏨 [Hotel Reservation REST API](https://github.com/mohammadpakdoust/hotel-api-project)
*   Developed a backend REST API using clean engineering design and database integration to handle bookings and reservations.
